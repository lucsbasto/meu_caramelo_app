# Meu Caramelo (MVP UI)

UI inicial em React Native + Expo com mapa, markers de comedouros e bottom sheet com carrossel.

## Como rodar

```bash
npm install
npm run start
```

Depois:
- abra o app **Expo Go** no iPhone
- escaneie o QR Code exibido no terminal/browser

## Stack usada
- Expo + React Native + TypeScript
- `react-native-maps`
- `@gorhom/bottom-sheet`
- `react-native-reanimated-carousel`
- `react-native-reanimated`
- `react-native-gesture-handler`
